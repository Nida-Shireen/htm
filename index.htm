WHAT IS HTML?

  1. HTML stands for Hyper Text Markup Language.
  2. HTML is the standard markup language for creating Web pages.
  3. HTML describes the structure of a Web page, helps to add content.
  4. HTML consists of a series of elements.
  5. HTML elements tell the browser how to display the content.
  6. any text editor can be used to run a html script

The ' < ! DOCTYPE > 'declaration represents the document type, and helps browsers to display web pages correctly. hence should be typed out at the begining of a html script
every html script should begin and end with a html tag
html has elements* and attributes.

*elements are tags.

tags are always and I mean always placed inside '< >'. and should be closed with '< / >'
